For the design of the frontend, you must use the following rules:

- Respect our graphic charter. (Documentation/Charte/Charte.md)
- Use the Tailwind CSS framework for styling.
- Use the React Router for navigation.
- Use the React Query for data fetching.
- Use the React Hook Form for form handling.
- Use the React Toastify for notifications.
- Use the React Icons for icons.
- Use the React Loader Spinner for loading states.
- Use the React Pagination for pagination.
- Use the React Select for select fields.
- Use the React Datepicker for date pickers.
- Use the React Timepicker for time pickers.
